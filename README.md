# Shubham Kumar - Personal Portfolio

Welcome to the repository for my personal portfolio website! This website showcases my skills, projects, achievements, and educational background as a Data Science student passionate about machine learning, data analytics, and building intelligent data-driven systems.

## 🚀 Features

- **Modern & Responsive Design:** A fully responsive layout that looks great on all devices (desktop, tablet, and mobile).
- **Interactive UI Elements:** 
  - Dynamic **Theme Switcher** with multiple color palettes (Deep Space, Sunset Glow, Ocean Breeze, Midnight Blue, Neon Forest, Cyberpunk, Rose Gold, Lavender). Keyboard shortcut `Ctrl + K` supported!
  - Interactive **Particle Background** for a modern tech feel.
  - Smooth **Scroll Animations** as you navigate through different sections.
  - Typing effects and hover animations.
- **Dedicated Sections:**
  - **Hero Section:** Introduction, brief bio, and a downloadable CV.
  - **Skills:** Categorized skills in Programming, Data Science, Machine Learning, Tools, and Core CS with visual progress donuts.
  - **Projects:** Highlighted Data Science and Software Development projects with direct links to the code.
  - **Education:** Timeline of my academic journey.
  - **Achievements & Certifications:** Showcase of my coding profiles (LeetCode, GFG) and certifications.
  - **Contact:** Easy ways to get in touch.

## 🛠️ Tech Stack

This portfolio is built using core web technologies without any heavy frameworks:
- **HTML5:** Semantic structuring.
- **CSS3:** Advanced styling, CSS variables for theme switching, Flexbox/Grid layouts, and custom animations.
- **JavaScript (Vanilla):** DOM manipulation, theme switching logic, scroll reveals, and integration of the particle background.
- **External Libraries:** 
  - [FontAwesome](https://fontawesome.com/) for icons.
  - [Devicon](https://devicon.dev/) for programming language icons.
  - Particles.js (if used).

## 💻 Running Locally

To view this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/ShubhamKumar284/<your-repo-name>.git
   ```
2. Navigate to the project directory:
   ```bash
   cd <your-repo-name>
   ```
3. Open `index.html` in your favorite web browser. You can also use a tool like Visual Studio Code's "Live Server" extension for a better development experience.
